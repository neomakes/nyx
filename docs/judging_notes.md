# Judging Notes

Nyx is meant to be judged as an interface pattern:

| Question | Expected Answer |
|---|---|
| Does it run locally? | Yes, with Python's standard library. |
| Does it need cloud access? | No. |
| Does it over-claim? | It should not. |
| What is the memorable behavior? | It tells the operator to recover confidence before moving deeper. |

The demo is small on purpose. The hard part is not the page; it is the refusal
to turn weak evidence into confident language.
