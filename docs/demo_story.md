# Demo Story

## Setup

An operator enters an indoor route where GPS, comms, and cloud access are not
available. A small local assistant tracks the route by mission packet,
checkpoint freshness, and observed cues.

## Moment

The operator asks:

```text
Can I continue?
```

Nyx does not answer with confidence it does not have.

```text
RE-ORIENT / REALIGN
```

## Reveal

The useful behavior is not prediction theater. The useful behavior is restraint.

```text
The card prevents a weak signal from becoming a strong claim.
```
