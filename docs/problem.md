# Problem

Signal-denied spaces turn simple questions into risky ones.

```text
Where am I?
Can I keep moving?
Can I trust this checkpoint?
What should I not claim yet?
```

Nyx is a small demo for that moment. It compresses weak local observations into
a conservative action card. The card is intentionally plain: it asks the human
to re-orient, realign, hold, or hand off instead of inventing certainty.

## Design Rule

The interface should be more useful than a chatbot and less ambitious than a
command system.

```text
Say less.
Over-claim less.
Keep the human oriented.
```
