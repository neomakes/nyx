> GPS가 끊기고, 통신이 약해지고, 공간이 반복될 때  
> Nyx는 대원에게 “계속 가라”가 아니라 **다시 확인할 기준점**을 줍니다.

Nyx는 로컬에서 통신없이 작동하는 온디바이스 AI 솔루션입니다. GPS, 통신,
클라우드 접근, 현장 확신이 동시에 약해지는 상황에서 대원이 방향감과 판단 기준을
잃지 않도록 돕습니다.

Nyx는 전술 지휘 체계가 아닙니다. 사람, 위협, 무기, 위험물, 진입 가능 여부를
식별하거나 확정하지 않습니다. 이 공개 레포는 **신호가 약한 공간에서 과신을 막는
인터페이스 패턴**을 보여주는 D4D 제출용 데모입니다.

```text
약한 증거 -> 보수적 상태 판단 -> 짧은 행동 카드 -> 사람의 최종 판단
```

## 핵심 내용 1 - 문제

현장 대원이 가장 위험해지는 순간은 정보가 없는 순간이 아니라, 약한 정보를 확신으로 착각하는 순간입니다.

지하 터널, 도시 지하, 참호, 폐건물, 재난 현장에서는 GPS, 통신, 시야, 지도 신뢰도가 동시에 흔들립니다. 이때 필요한 것은 더 긴 설명이 아니라 **무엇을 확정하지 말아야 하는지**와 **무엇을 다시 확인해야 하는지**입니다.

## 핵심 내용 2 - 사용자

Nyx가 보는 사용자는 압박 속에서 짧게 판단해야 하는 팀입니다.

| 사용자 | 페인포인트 |
|---|---|
| 분대장 / 팀 리더 | 마지막 체크포인트가 낡았는데 계속 전진해도 되는지 판단하기 어렵습니다. |
| 공병 / 수색 인원 | 단서 하나만 보고 안전, 위협, 목표 도달을 과잉 확정하기 쉽습니다. |
| 상황실 / 인계 담당 | 무엇이 사실이고 무엇이 추정인지 분리되지 않아 인계 품질이 떨어집니다. |

## 핵심 내용 3 - 솔루션

Nyx는 복잡한 작전 명령을 만들지 않습니다. 현장 화면을 네 가지 행동 경계로 줄입니다.

```text
RE-ORIENT  마지막으로 믿을 수 있는 기준점을 회복
REALIGN    경로와 관찰 단서가 어긋나면 재정렬
HOLD       증거가 약하면 확정 금지
HANDOFF    다른 팀이나 자산으로 인계
```

Nyx가 발전시키는 지점은 AI의 정답 생성이 아니라 **불확실성을 화면의 행동 경계로 바꾸는 것**입니다.
