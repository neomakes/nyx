> GPS가 끊기고, 통신이 약해지고, 공간이 반복될 때  
> Nyx는 대원에게 “계속 가라”가 아니라 **현재 상태를 다시 잡을 기준**을 줍니다.

Nyx는 GPS와 통신이 끊긴 지하, 도심, 참호 작전에서 대원이 길, 접촉, 인계 판단을 잃지 않게 하는 온디바이스 AI 코파일럿입니다.

Nyx는 전술 지휘 체계가 아닙니다. 사람, 위협, 무기, 위험물, 진입 가능 여부를
식별하거나 확정하지 않습니다. 이 공개 레포는 **신호가 약한 공간에서 과신을 막는
인터페이스 패턴**을 보여주는 D4D 제출용 데모입니다.

```text
약한 증거 -> 보수적 상태 판단 -> 짧은 행동 카드 -> 사람의 최종 판단
```

## 핵심 내용 1 - 문제

현대전은 로봇을 먼저 넣는 방향으로 발전하고 있지만, 지하, 시가전, 참호전은 완전 무인화로 닫히지 않습니다. 통로는 좁고, 전개는 빠르며, 통신은 끊기고, 마지막 확인은 결국 현장 대원에게 돌아옵니다.

이때 가장 위험한 순간은 정보가 없는 순간이 아니라, 약한 정보를 확신으로 착각하는 순간입니다. 반복되는 계단과 복도, 무너진 구획, 참호의 굴곡, 열원 하나가 사람인지 물체인지 모호한 상황에서 지도와 기억은 빠르게 낡습니다.

기존 방식에서는 야간/열상 관측, 위치·지도 확인, 거리·장애물 감지, 무전 보고, 로봇·전문팀 인계가 서로 다른 장비와 절차로 흩어집니다. 공개 장비만 보아도 열상/야간 융합 고글은 헬멧에 붙고, 다중 채널 무전기는 배터리 포함 약 2.75lb급이며, 소형 지상로봇도 수십 lb급 장비가 됩니다. Nyx 데모는 약 1.5kg, 25 x 15 x 7cm 내외의 착용형 폼팩터에서 이 관찰들을 한 화면의 상태 판단으로 묶는 방향을 보여줍니다.

문제는 장비가 없다는 것이 아니라, 급박한 현장에서 이 단서들을 다시 조합해야 한다는 것입니다.

이때 필요한 것은 더 긴 설명이 아니라 **내 위치와 방향이 아직 믿을 만한지**, **접촉을 확정해도 되는지**, **어느 지점에서 멈추고 인계해야 하는지**를 한 화면에서 붙잡는 것입니다.

## 핵심 내용 2 - 사용자

Nyx가 보는 사용자는 지하, 도심, 참호처럼 신호가 약하고 공간이 닫힌 작전 환경에서 직접 움직이는 전방 대원입니다.

| 작전 사용자 | 현장 페인포인트 | Nyx가 줄이는 부담 |
|---|---|---|
| 지하/도심 구획실 수색 대원 | 한 손에는 무전, 시야에는 열상/야간 장비, 머릿속에는 마지막 체크포인트가 따로 놀면서 방향감이 무너집니다. | 체크포인트, 진행 방향, 관찰 단서를 한 상태 화면으로 압축합니다. |
| 참호/엄폐 공간 이동 대원 | 드론, 재밍, 가림, 우회로 때문에 몇 초 안에 이동, 정지, 확인을 다시 판단해야 합니다. | 경로 어긋남과 의심 지점을 표시하고 확정 금지를 유지합니다. |
| 공병/EOD/로봇 연계 대원 | 로봇을 보낼 수 있어도 모든 구간을 로봇만으로 처리하기 어렵고, 미확인 열원과 물체의 의미를 현장에서 분기해야 합니다. | 표시, 이격, 확인, 로봇 탐색, 전문팀 인계를 짧은 행동 카드로 정리합니다. |

## 핵심 내용 3 - 솔루션

Nyx는 복잡한 작전 명령을 만들지 않습니다. 여러 장비와 절차에 흩어진 관찰을 로컬에서 현장 상태로 압축하고, 그 상태를 짧은 행동 카드로 바꿉니다.

```text
RGB/depth + thermal + ultrasonic + IMU
        -> field state
        -> action card
```

데모에서 Nyx의 카드는 다음처럼 동작합니다.

| 카드 | 의미 |
|---|---|
| ORIENT / TRACK | 위치, 방향, 깊이, 마지막 체크포인트를 계속 붙잡습니다. |
| RE-ORIENT / REALIGN | 경로와 랜드마크가 어긋나면 기준점을 회복하고 재정렬합니다. |
| MARK / STANDOFF | 고열 물체나 의심 접촉은 표시하고 이격하며 안전 확정을 금지합니다. |
| CALL OUT / CONFIRM | 사람형 가능성이 있어도 신원과 역할을 확정하지 않고 확인 루프로 보냅니다. |
| ROBOT-PROBE / HANDOFF | 위험이 해소되지 않으면 로봇, 전문팀, 다른 팀으로 인계합니다. |

Nyx가 발전시키는 지점은 AI의 정답 생성이 아니라 **센서 관찰을 보수적 현장 상태와 행동 경계로 바꾸는 온디바이스 루프**입니다.
