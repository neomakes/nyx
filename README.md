> GPS가 끊기고, 통신이 약해지고, 공간이 반복될 때  
> Nyx는 대원에게 “계속 가라”가 아니라 **현재 상태를 다시 잡을 기준**을 줍니다.

Nyx는 로컬에서 통신없이 작동하는 온디바이스 AI 솔루션입니다. GPS, 통신,
클라우드 접근, 현장 확신이 동시에 약해지는 상황에서 대원이 방향감과 판단 기준을
잃지 않도록 돕습니다.

Nyx는 전술 지휘 체계가 아닙니다. 사람, 위협, 무기, 위험물, 진입 가능 여부를
식별하거나 확정하지 않습니다. 이 공개 레포는 **신호가 약한 공간에서 과신을 막는
인터페이스 패턴**을 보여주는 D4D 제출용 데모입니다.

```text
약한 증거 -> 보수적 상태 판단 -> 짧은 행동 카드 -> 사람의 최종 판단
```

## 핵심 내용 1 - 문제

현장 대원이 가장 위험해지는 순간은 정보가 없는 순간이 아니라, 약한 정보를 확신으로 착각하는 순간입니다.

지하 터널, 도시 구획실, 참호, 폐건물에서는 GPS, 통신, 시야, 지도 신뢰도가 동시에 흔들립니다. 통로는 반복되고, 열원은 사람인지 물체인지 애매하며, 마지막 기준점도 빠르게 낡습니다.

기존 방식에서는 야간/열상 관측, 위치·지도 확인, 거리·장애물 감지, 무전 보고, 로봇·전문팀 인계가 서로 다른 장비와 절차로 흩어집니다. Nyx가 줄이려는 부담은 장비 자체가 아니라, 그 장비들이 만든 약한 단서를 현장에서 다시 해석해야 하는 부담입니다.

이때 필요한 것은 더 긴 설명이 아니라 **내 위치와 방향이 아직 믿을 만한지**, **접촉을 확정해도 되는지**, **어느 지점에서 멈추고 인계해야 하는지**를 한 화면에서 붙잡는 것입니다.

## 핵심 내용 2 - 사용자

Nyx가 보는 사용자는 지하, 도심, 참호처럼 신호가 약하고 공간이 닫힌 작전 환경에서 움직이는 팀입니다.

| 작전 사용자 | 현장 페인포인트 | Nyx가 줄이는 부담 |
|---|---|---|
| 지하/도심 구획실 수색팀 | 반복 통로, 낮은 시야, 미확인 접촉으로 방향감이 무너집니다. | 체크포인트, 진행 방향, 관찰 단서를 짧은 상태로 압축합니다. |
| 참호/엄폐 공간 이동팀 | 재밍, 가림, 열원 접촉 속에서 경로 이탈과 과잉 확정이 같이 생깁니다. | 의심 지점과 경로 어긋남을 표시하고 확정 금지를 유지합니다. |
| 공병/EOD/로봇 연계팀 | 고열 물체, 가림, 부비트랩 의심을 현장에서 바로 확정하기 어렵습니다. | 표시, 이격, 확인, 로봇 탐색, 전문팀 인계를 행동 카드로 정리합니다. |

## 핵심 내용 3 - 솔루션

Nyx는 복잡한 작전 명령을 만들지 않습니다. 여러 장비와 절차에 흩어진 관찰을 로컬에서 현장 상태로 압축하고, 그 상태를 짧은 행동 카드로 바꿉니다.

```text
RGB/depth + thermal + ultrasonic + IMU
        -> field state
        -> action card
```

데모에서 Nyx의 카드는 다음처럼 동작합니다.

| 카드 | 의미 |
|---|---|
| ORIENT / TRACK | 위치, 방향, 깊이, 마지막 체크포인트를 계속 붙잡습니다. |
| RE-ORIENT / REALIGN | 경로와 랜드마크가 어긋나면 기준점을 회복하고 재정렬합니다. |
| MARK / STANDOFF | 고열 물체나 의심 접촉은 표시하고 이격하며 안전 확정을 금지합니다. |
| CALL OUT / CONFIRM | 사람형 가능성이 있어도 신원과 역할을 확정하지 않고 확인 루프로 보냅니다. |
| ROBOT-PROBE / HANDOFF | 위험이 해소되지 않으면 로봇, 전문팀, 다른 팀으로 인계합니다. |

Nyx가 발전시키는 지점은 AI의 정답 생성이 아니라 **센서 관찰을 보수적 현장 상태와 행동 경계로 바꾸는 온디바이스 루프**입니다.
