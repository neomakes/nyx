> GPS가 끊기고, 통신이 약해지고, 공간이 반복될 때  
> Nyx는 대원에게 “계속 가라”가 아니라 **현재 상태를 다시 잡을 기준**을 줍니다.

Nyx는 GPS와 통신이 끊긴 지하, 도심, 참호 작전에서 대원이 길, 접촉, 인계 판단을 잃지 않게 하는 온디바이스 AI 코파일럿입니다.

Nyx는 전술 지휘 체계가 아닙니다. 사람, 위협, 무기, 위험물, 진입 가능 여부를
식별하거나 확정하지 않습니다. 이 공개 레포는 **신호가 약한 공간에서 과신을 막는
인터페이스 패턴**을 보여주는 D4D 제출용 데모입니다.

```text
약한 증거 -> 보수적 상태 판단 -> 짧은 행동 카드 -> 사람의 최종 판단
```

## 핵심 내용 1 - 문제

```text
로봇만으로 닫히지 않는 공간: 지하 / 시가전 / 참호
흩어진 장비: 열상·야간 + 지도·위치 + 거리 감지 + 무전 + 로봇 인계
실제 부담: 장비가 아니라, 급박한 순간에 약한 단서를 다시 조합하는 일
```

Nyx 데모는 약 1.5kg, 25 x 15 x 7cm 내외의 착용형 폼팩터에서 이 관찰들을 한 화면의 상태 판단으로 묶는 방향을 보여줍니다.

## 핵심 내용 2 - 사용자

| 현장 대원 | 3초 안에 겪는 문제 | Nyx가 남기는 것 |
|---|---|---|
| 구획실 수색 대원 | 반복 통로에서 마지막 체크포인트가 흐려짐 | 현재 방향 + 기준점 |
| 참호/엄폐 이동 대원 | 드론, 재밍, 우회로 속에서 정지/이동 판단이 흔들림 | 경로 어긋남 + 확정 금지 |
| 공병/EOD/로봇 연계 대원 | 로봇을 보낼지, 사람이 확인할지, 인계할지 갈림 | 표시 + 이격 + 인계 카드 |

## 핵심 내용 3 - 솔루션

```text
RGB/depth + thermal + ultrasonic + IMU  ->  field state  ->  action card
길을 잃으면 ORIENT / REALIGN
접촉이 애매하면 MARK / CONFIRM / HANDOFF
```

| 카드 | 의미 |
|---|---|
| ORIENT / REALIGN | 기준점 회복, 경로 재정렬 |
| MARK / CONFIRM | 의심 접촉 표시, 사람의 확인 유지 |
| STANDOFF / HANDOFF | 이격, 로봇·전문팀·다른 팀 인계 |

Nyx의 핵심은 정답 생성이 아니라 **관찰을 보수적 행동 경계로 바꾸는 온디바이스 루프**입니다.
